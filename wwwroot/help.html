GET: Maths Endpoint

List of possible query strings:

  • Addition: ?op=+&x=[value]&y=[value]
    Returns: {"op":"+", "x":[value], "y":[value], "result":[x + y]}
  • Subtraction: ?op=-&x=[value]&y=[value]
    Returns: {"op":"-", "x":[value], "y":[value], "result":[x - y]}
  • Multiplication: ?op=*&x=[value]&y=[value]
    Returns: {"op":"*", "x":[value], "y":[value], "result":[x * y]}
  • Division: ?op=/&x=[value]&y=[value]
    Returns: {"op":"/", "x":[value], "y":[value], "result":[x / y]}
    Note: y must not be 0 to avoid division by zero error.
  • Modulo: ?op=%&x=[value]&y=[value]
    Returns: {"op":"%", "x":[value], "y":[value], "result":[x % y]}
  • Factorial: ?op=!&n=[value]
    Returns: {"op":"!", "n":[value], "result":[n!]}
    Note: n must be a non-negative integer.
  • Prime Check: ?op=p&n=[value]
    Returns: {"op":"p", "n":[value], "isPrime":[true/false]}
    Note: n must be a positive integer.
  • Find nth Prime: ?op=np&n=[value]
    Returns: {"op":"np", "n":[value], "result":[nth prime number]}
    Note: n must be a positive integer.
